1.2.9.7 SYS_PORT_PinInputEnable Function

C

void SYS_PORT_PinInputEnable(SYS_PORT_PIN pin)

Summary

Enables selected IO pin as input.

Description

This function enables selected IO pin as input.

Precondition

None.

Parameters

ParamDescription
pinOne of the IO pins from the enum SYS_PORT_PIN

Returns

None.

Example

SYS_PORT_PinInputEnable(SYS_PORT_PIN_PB3);

Remarks

None.
