Building the Application

The parent folder for all the projects for this application is given below:

Application Path 
expresslogic_threadx\apps\basic\firmware 

 

To build the application, refer to the following table and open the appropriate project file. 

 

MPLAB X IDE Projects

Project Name 
Description 
pic32mk_mcj_curiosity_pro.X 
PIC32MK MCJ Curiosity Pro Development Board 
pic32mz_das_sk.X 
PIC32MZ Embedded Graphics with Stacked DRAM (DA) Starter Kit (Crypto) 
pic32mz_ef_sk.X 
PIC32MZ Embedded Connectivity with FPU (EF) Starter Kit 
sam_c21n_xpro.X 
SAM C21N Xplained Pro Evaluation Kit 
sam_e54_xpro.X 
SAM E54 Xplained Pro Evaluation Kit

Project Name 
Description

SAM9X60 evaluation kit 

 

Note:
  1. Make sure the Express Logic ThreadX source code is installed in <Harmony3_Path>/thirdparty_expresslogic/tx58<core_name>_<ide_name> then regenerate the code for demonstration to work. Ex: Express Logic ThreadX Source Code for Cortex-M7 based device is located at <Harmony3_Path>/thirdparty_expresslogic/tx58cortex-m7_mplabx
  2. The following IAR EW Project configurations must be performed before building the IAR EW Project.
    • Go to IAR EW IDE Project -> Options... -> C/C++ Compiler -> Code -> Processor Mode, and Select Arm Processor mode.
    • Go to IAR EW IDE Project -> Options... -> C/C++ Compiler -> Preprocessor ->Defined symbols: (one per line), and add TX_INCLUDE_USER_DEFINE_FILE in the text box.
