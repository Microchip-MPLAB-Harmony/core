Introduction

This driver library provides application ready routines to read and write to the SD Card, thus minimizing developerâ€™s awareness of working of the SD Card protocol.

  • The driver is currently supported only in RTOS environment
  • Provides synchronous (blocking) mode operation
  • Supports both DMA and non-DMA modes
  • The driver API interface can be used directly by the application to read/write directly from the SD Card Or the SDSPI driver can be used with a file system
